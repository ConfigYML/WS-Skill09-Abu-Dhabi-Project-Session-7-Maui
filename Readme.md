# Starting the application
To start the application please ensure that the API is running on 'http://localhost:5000'.
You can then either start the application by opening visual studio and running it with a phone emulator of your choice or you can install the .APK file located under the following path: 'Executable\net9.0-android\publish\com.companyname.session7dennishilfinger-Signed.apk'
